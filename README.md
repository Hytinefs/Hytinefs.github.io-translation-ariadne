>你们达不到这一步，除非有真情实感，除非它发自心灵，并以充满原始力量的快意去打动所有听众的心。可你们尽管坐着！粘粘贴贴，拼拼凑凑，用别人的残羹剩汁去烩一碗佳肴，从你们一小堆灰烬里吹出寒伧的火苗！可如果你有这种嗜好，未尝不可博得儿童和愚人们的赞颂——可你们永远做不到心心相印，如果你们言不由衷。
><p align="right"> ——《浮士德》</p>
